# KeshaGame
 The Best Game In The Whole World!)))
 
 NEED TO KNOW:

 Start chunk is C_300_900

 AllFather.cs is father for all static classes

 S.cs is main static class. You can get every static thing by S.

Initialiser.cs puts links into S

Game now contains 6 x 6 chunks with repeating

Item.cs is for inventory.
ItemP.cs is for items in the World

Items can be throwed only when they have prefab in Assets/Resources/Prefabs

Name of the item's prefab and sprite MUST be placed in II.cs (Which is ItemInfo);

Items MUST be on NotTransperent layer, otherwise they will be unclickable

Day and Night time now is 10 minutes

The game starts from the Start scene. Start scene never unloads.
